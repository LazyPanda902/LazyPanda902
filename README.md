## Featured Projects

### ad-health-checker
Python CLI for Active Directory DNS, Kerberos, LDAP, and service-port health checks.

- Raw socket-based service checks
- LDAP listener probe with structured reporting
- JSON and text output for troubleshooting
- CI-tested across Python versions

### supportops-toolkit
Python CLI for IT account auditing and SSH authentication log analysis.

- Audits exported account CSVs for risky patterns
- Detects repeated login failures and brute-force behavior
- Produces structured JSON reports
- Built for help desk and junior sysadmin workflows

### homelab-healthwatch
Zero-dependency Python CLI for Raspberry Pi homelab container and service health checks.

- Docker container status checks
- HTTP endpoint probing
- Local system metrics from `/proc`
- Useful for lightweight self-hosted monitoring
